# Java Basics ☕

This repository contains beginner-friendly Java programs created as part of my Java Developer learning journey.

## 📚 Programs Included

### 1. HelloWorld.java

A simple program to understand the basic structure of a Java application and print text to the console.

**Concepts Covered:**

* Java program structure
* `main()` method
* `System.out.println()`

**Sample Output:**

```text
Hello, World!
This is my first Java program.
Excited to begin my Java Developer journey!
```

### 2. Variables.java

**Concepts Covered:**

* Variables
* Primitive data types (`int`, `double`, `float`, `char`, `boolean`, `long`, `short`, `byte`)
* `String`
* Printing variable values
